# コードクリーンアップチェック

サブエージェントが使える環境では第三者視点でレビューさせ、使えない環境では同じ観点で自己レビューする。
差分の設計妥当性に不安がある場合は `consult` または `review-loop` へ。

## サブエージェント契約

- レビュー中は変更しない。適用・検証・報告は親
- harness 固有の agent API は指定しない
- 使えない環境では同じチェックリストで自己レビュー

## 起点・調査・修正

| 軸 | 意味 |
| --- | --- |
| 起点 | 今回触ったファイル |
| 調査 | 周辺・caller・兄弟を read-only で読む。制限しない |
| 修正 | 起点に加え、見つけた改善も積極的に直す |
