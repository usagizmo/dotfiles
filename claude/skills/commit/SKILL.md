commit して。言語は1つ前のコミットを参考にして、先頭には適切な gitmoji をつけて。

## 形式

```
{gitmoji} {message}

- {詳細1}
- {詳細2}
- ...
```

## gitmoji

gitmoji の選び方は `references/gitmoji.md` を参照。
